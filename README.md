
# Coding language version control
```zsh

```

# asdf syntax
```zsh
asdf plugin add neovim (java,python,poetry,nodejs,yarn,etc.)
asdf install neovim latest
asdf set -u neovim {latest}
asdf list
```

## Commands
`touch ..` --> Create file
`mkdir ..` --> Create folder
`rm ...` --> delete file
`rm -r ..` --> delete folder
